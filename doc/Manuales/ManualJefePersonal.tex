\chapter{Introducción.} 

\section{Propósito.}

\indent \indent En este documento se presenta la funcionalidad básica de SEPROSO desde el punto de vista de un jefe de personal que utiliza la herramienta. El objetivo de este manual es proporcionar al jefe de proyecto la información necesaria para utilizar la herramienta SEPROSO.


\section{Ámbito.}

\indent \indent Mediante esta práctica herramienta Web se pretender dar capacidad para:

\begin{itemize}
	\item Planificar sus proyectos software.
  \item Seguir y monitorizar el desarrollo del proyecto.
  \item Controlar y aprobar los informes de trabajo de los desarrolladores.
  \item Conocer en cada momento la situación de un trabajador.
\end{itemize}

\chapter{Login.}

\indent \indent El login del \textbf{Jefe de Personal} del sistema se realiza desde la pantalla de inicio de la herramienta SEPROSO. A continuación se muestra la pantalla inicial del sistema \ref{log} y que sirve de Login para cualquier tipo de usuario. El jefe de personal solamente se debe de encargar de introducir un password correcto asociado a la cuenta de usuario cuyo rol sea jefe de personal.\\

\figura{Images/login.png}{width=0.9\linewidth}{Login de la herramienta SEPROSO.}{log}

En caso de introducir una contraseña errónea se mostrará al usuario el siguiente mensaje de error  \ref{log_e}.\\

\figura{Images/Login_errno.png}{width=0.9\linewidth}{Mensaje de error ante un login inválido.}{log_e}

Una vez conseguido el acceso a la herramienta se recibe un mensaje de bienvenida, identificándose cómo un usuario determinado por los menús superiores que se le presentan a cada usuario \ref{bienvenida}. En el caso del Jefe de personal se le muestran los menús de configuración de Situación de Personal e Informe de ocupación. Se puede observar en la siguiente figura \ref{men_admin}.\\

\figura{Images/Bienvenido.png}{width=0.9\linewidth}{Mensaje de bienvenida a la herramienta.}{bienvenida}

\figura{Images/menu_jp.png}{width=0.9\linewidth}{Menu de edición del administrador del sistema.}{men_admin}
