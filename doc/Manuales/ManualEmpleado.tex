\chapter{Introducción.} 

\section{Propósito.}

\indent \indent En este documento se presenta la funcionalidad básica de SEPROSO desde el punto de vista de un desarrollador que utiliza la herramienta. El objetivo de este manual es proporcionar al usuario la información necesaria para utilizar la herramienta SEPROSO.


\section{Ámbito.}

\indent \indent Mediante esta práctica herramienta Web se pretender dar capacidad para

\begin{itemize}
	\item Planificar sus proyectos software.
  \item Seguir y monitorizar el desarrollo del proyecto.
  \item Controlar y aprobar los informes de trabajo de los desarrolladores.
  \item Conocer en cada momento la situación de un trabajador.
\end{itemize}

\indent  \indent Véase el Glosario.

\chapter{Login.}

\indent \indent El login de cualquier \textbf{empleado} del sistema se realiza desde la pantalla de inicio de la herramienta SEPROSO. A continuación se muestra la pantalla inicial del sistema \ref{log} y que sirve de Login para cualquier tipo de usuario. Un empleado cuyo rol no sea específico solamente se debe de encargar de introducir un password correcto asociado a la cuenta de usuario.\\

\figura{Images/login.png}{width=0.9\linewidth}{Login de la herramienta SEPROSO.}{log}

En caso de introducir una contraseña errónea se mostrará al usuario el siguiente mensaje de error  \ref{log_e}.\\

\figura{Images/Login_errno.png}{width=0.9\linewidth}{Mensaje de error ante un login inválido.}{log_e}

Una vez conseguido el acceso a la herramienta se recibe un mensaje de bienvenida, identificándose cómo un usuario determinado por los menús superiores que se le presentan a cada usuario \ref{bienvenida}. En el caso de un desarrollador normal se le muestran los menús Actividades, Informes de Actividad y Configuración de vacaciones. Se puede observar en la siguiente figura \ref{men_admin}.\\

\figura{Images/Bienvenido.png}{width=0.9\linewidth}{Mensaje de bienvenida a la herramienta.}{bienvenida}

\chapter{Registrar informe de Actividad.}

\indent \indent Cualquier desarrollador es capaz de registrar el informe de una actividad que tiene asociada. Para ello debe acceder a la herramienta y entrar en la opción \textit{ACTIVIDAD}. Una vez allí le aparecerá la plantilal de registro de actividad para registrar el informe de una actividada ya comenzada.\\

Se debe indicar la actividad realizada, la fecha, el número de horas implicado en ella y un breve cometnario que es opcional. Podemos ver la plantilla en la imagen \ref{plantilla}. Una vez acabado se pulsará sobre la opción de registrar informe.\\

\figura{Images/reg_act.png}{width=0.9\linewidth}{Plantilla de registro de actividad.}{plantilla}
