\section{Ámbito.}

\indent \indent Mediante esta práctica herramienta Web se pretender dar capacidad para

\begin{itemize}
	\item Planificar sus proyectos software.
  \item Seguir y monitorizar el desarrollo del proyecto.
  \item Controlar y aprobar los informes de trabajo de los desarrolladores.
  \item Conocer en cada momento la situación de un trabajador.
\end{itemize}

\chapter{Instalación.}

\indent \indent El modo de instalación de la herramienta es sencillo, basta con la ejecución en modo de comandos de un Script preparado para la instalación. El Script se añade con la práctica para su instalación, se debe usar el siguiente código:\\

\indent \indent \indent \indent \textit{instalar.sh DIR DATABASE USER PASSWORD}\\

donde:

\begin{itemize}
	\item Se copia en la dirección indicada por DIR la configuración web de la herramienta y el framework utilizado para su desarrollo.
	\item Se crea un fichero de configuración de la base de datos con una conexión a la base de datos DATABASE para el USUARIO y PASSWORD indicados.
\end{itemize}
